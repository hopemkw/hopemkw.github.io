### 介绍
一款支持支持网易、酷狗、酷我、百度和咪咕音乐的播放器。

### 主要功能：
1. 以上平台的歌单及歌单内的歌曲播放，包括热门歌单、分类歌单等
2. 以上平台的音乐的搜索及播放，但不包括收费歌曲。
3. 支持以上平台的排行榜歌曲及播放。
4. 歌曲收藏，后面会加上歌单收藏。
5. 自动适配系统浅色和暗色主题，也可以手动切换。
